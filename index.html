URL Unduhan Permanen

URL ini selalu mengarah ke aset installer terbaru. Cocok dibagikan ke operator baru atau disimpan oleh koordinator.

Panduan Pemasangan

Ikuti langkah berikut setelah file installer selesai diunduh.

1

Unduh installer

Klik tombol Unduh Sekarang. File SIGAP-Setup.exe akan tersimpan di folder Downloads.

2

Jalankan file

Buka file dengan klik dua kali. Jika Windows menampilkan peringatan, pilih opsi untuk tetap menjalankan.

3

Pilih lokasi

Ikuti panduan installer. Lokasi pemasangan bisa memakai bawaan atau folder lain sesuai kebutuhan satker.

4

Selesai

Buka SIGAP dari Desktop atau Start Menu. Aplikasi dapat digunakan tanpa koneksi internet.

i
PC operator tidak terhubung internet?

Koordinator dapat mengunduh installer dari PC yang memiliki internet, lalu membawa file melalui flashdisk.

1Unduh SIGAP-Setup.exe dari halaman ini.
2Salin file installer ke flashdisk.
3Jalankan installer di PC operator.
4Untuk pembaruan manual berikutnya, gunakan menu pengaturan aplikasi jika tersedia.
u
Pembaruan otomatis setelah aplikasi terpasang

Jika PC memiliki internet, SIGAP akan mengecek rilis terbaru di latar belakang. Operator tetap harus menyetujui proses unduh dan pemasangan, sehingga tidak ada perubahan diam-diam.
