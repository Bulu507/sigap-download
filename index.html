Panduan Pemasangan

Ikuti langkah berikut setelah file installer selesai diunduh.

1

Unduh installer

Klik tombol Unduh Sekarang. File seperti SIGAP-Setup-0.1.7.exe akan tersimpan di folder Downloads.

2

Jalankan file

Buka file dengan klik dua kali di Windows 10 atau Windows 11 64-bit. Jika Windows menampilkan peringatan, pilih opsi untuk tetap menjalankan.

3

Pilih lokasi

Ikuti panduan installer. Lokasi pemasangan bisa memakai bawaan atau folder lain sesuai kebutuhan satker.

4

Selesai

Buka SIGAP dari Desktop atau Start Menu. Aplikasi menyimpan data lokal dan dapat digunakan tanpa koneksi internet.

i
PC operator tidak terhubung internet?

Koordinator dapat mengunduh installer dari PC yang memiliki internet, lalu membawa file melalui flashdisk ke PC operator Windows 10 atau Windows 11.

1Unduh file SIGAP-Setup-[versi].exe dari halaman ini.
2Salin file installer ke flashdisk.
3Jalankan installer di PC operator.
4Untuk pembaruan manual berikutnya, unduh installer terbaru dari halaman ini dan jalankan kembali di PC operator.
u
Pembaruan otomatis setelah aplikasi terpasang

Jika PC memiliki internet, SIGAP akan mengecek rilis terbaru dari repo distribusi publik. Operator tetap harus menyetujui proses unduh dan pemasangan, sehingga tidak ada perubahan diam-diam.
