Panduan Pemasangan

Ikuti langkah berikut setelah file installer selesai diunduh.

1

Unduh installer

Klik tombol Unduh Sekarang. File seperti SIGAP-Setup-0.1.7.exe tersimpan di folder Downloads.

2

Jalankan file

Buka file installer di Windows 10 atau Windows 11 64-bit. Jika muncul peringatan Windows, pilih opsi untuk tetap menjalankan.

3

Ikuti installer

Pakai lokasi bawaan atau folder lain sesuai kebijakan satker. Proses pemasangan tidak membutuhkan data payroll.

4

Buka SIGAP

Jalankan dari Desktop atau Start Menu. Aplikasi dapat digunakan tanpa internet setelah terpasang.

i
PC operator tidak terhubung internet?

Koordinator dapat mengunduh installer dari PC yang memiliki internet, lalu membawa file melalui flashdisk ke PC operator.

1Unduh file SIGAP-Setup-[versi].exe dari halaman ini.
2Salin file installer ke flashdisk.
3Jalankan installer di PC operator.
4Untuk pembaruan manual, unduh installer terbaru lalu jalankan kembali.
u
Pembaruan otomatis setelah aplikasi terpasang

Jika PC memiliki internet, SIGAP dapat mengecek rilis terbaru dari repo distribusi publik. Operator tetap menyetujui proses unduh dan pemasangan.
